# 【数字电源/MATLAB+PLECS】如何进行 Buck 数字电源仿真（二）PLECS 搭建开环 Buck 功率级

上一篇主要介绍了 Buck 电路、数字电源控制链路以及本系列的整体路线。本文开始进入实操，在 PLECS 中先搭建一个开环 Buck 功率级模型。

这里先强调一个顺序：

```text
先开环
再闭环
先验证功率级
再设计控制器
```

不要一开始就急着加 PI 控制器。因为如果功率级模型本身接错了，或者电感、电容、负载参数不合理，后面调 PI 参数只会把问题掩盖得更深。

本文的目标是：

```text
固定占空比 duty
观察输出电压 Vout
观察电感电流 IL
观察开关节点 SW
确认 Buck 功率级模型符合基本规律
```

## 本篇要完成的内容

本文只做开环 Buck，不做闭环控制。

本篇完成后，模型中应该至少包含：

```text
输入电源 Vin
PWM 信号
高边开关管
续流二极管或同步开关
电感 L
输出电容 C
负载 R
电压采样
电流采样
Scope 波形观察
```

本篇不会加入：

```text
PI 控制器
软启动
过压保护
过流保护
状态机
C 代码
```

这些内容放到后续章节。这样做的原因是职责要清楚：本文只证明“被控对象”是正常的。

## 开环 Buck 的验证思路

理想 Buck 电路中，输出电压近似满足：

```text
Vout = D * Vin
```

其中：

| 名称 | 含义 |
| --- | --- |
| Vin | 输入电压 |
| Vout | 输出电压 |
| D | PWM 占空比 |

本系列标称输入电压为 24V，目标输出电压为 12V，所以开环测试时可以先取：

```text
D = 0.5
```

理论上：

```text
Vout = 0.5 * 24V = 12V
```

实际仿真中，如果考虑器件压降、电感电阻、电容 ESR，输出电压会略有偏差。第一版模型可以先从理想器件开始，确认拓扑正确后，再逐步加入非理想参数。

## 初始参数设计

先给出一组适合入门仿真的参数。

| 参数 | 初始值 | 说明 |
| --- | --- | --- |
| Vin | 24V | 标称输入电压 |
| Vout | 12V | 目标输出电压 |
| Iout | 5A | 最大输出电流 |
| Pout | 60W | 最大输出功率 |
| fsw | 200kHz | PWM 开关频率 |
| duty | 0.5 | 开环固定占空比 |
| L | 22uH | 初始电感值 |
| C | 100uF | 初始输出电容 |
| Rload | 2.4Ω | 满载等效负载 |

负载电阻的计算如下：

```text
Rload = Vout / Iout = 12V / 5A = 2.4Ω
```

电感值可以按电感电流纹波估算。假设电感电流纹波取满载电流的 30%：

```text
ΔIL = 5A * 30% = 1.5A
```

Buck 电感估算公式：

```text
L = (Vin - Vout) * D / (ΔIL * fsw)
```

代入参数：

```text
L = (24V - 12V) * 0.5 / (1.5A * 200kHz)
L ≈ 20uH
```

因此第一版可以选一个常见值：

```text
L = 22uH
```

输出电容先取：

```text
C = 100uF
```

这里不追求一次算得非常精确。第一版重点是把模型跑起来，并能解释波形。后续章节会再讨论电感、电容、纹波和动态响应之间的关系。

## PLECS 元件清单

在 PLECS 中，可以按下面的元件搭建第一版开环 Buck：

| 元件 | 作用 |
| --- | --- |
| DC Voltage Source | 输入电源 Vin |
| Controlled Switch 或 MOSFET | 高边开关管 |
| Diode | 续流二极管 |
| Inductor | Buck 电感 |
| Capacitor | 输出电容 |
| Resistor | 等效负载 |
| Pulse Generator | 产生固定 duty PWM |
| Voltage Sensor | 采样 Vout 和 SW 节点 |
| Current Sensor | 采样电感电流 IL |
| Scope | 观察仿真波形 |
| Electrical Reference | 电气参考地 |

如果刚开始不熟悉 PLECS 元件库，可以先使用理想开关和理想二极管。等开环模型验证正确后，再加入器件导通压降、导通电阻和电容 ESR。

## 搭建步骤

### 1. 新建 PLECS 模型

打开 PLECS，新建一个模型文件，建议命名为：

```text
buck_power_stage.plecs
```

后续可以放到 GitHub 仓库：

```text
models/plecs/buck_power_stage.plecs
```

截图位置：

```text
assets/screenshots/02-plecs-new-model.png
```

这里需要截一张 PLECS 新建空白模型的图，作为教程开始的实操截图。

### 2. 放置输入电源和参考地

先放置 DC Voltage Source，并设置：

```text
Vin = 24V
```

再放置 Electrical Reference，作为电路地。

这一步的目标是先建立输入电源和公共参考点。电源仿真中如果忘记参考地，模型很容易报错或者波形异常。

截图位置：

```text
assets/screenshots/02-plecs-vin-ground.png
```

### 3. 搭建高边开关和续流路径

Buck 的核心是开关管周期性导通和关断。

先放置高边开关管，再在 SW 节点到地之间放置续流二极管。

这里的 SW 节点非常重要，后续要观察它的波形：

```text
开关管导通时：SW 节点接近 Vin
开关管关断时：电感电流通过二极管续流，SW 节点电压下降
```

截图位置：

```text
assets/screenshots/02-plecs-switch-diode.png
```

### 4. 放置电感、输出电容和负载

在 SW 节点后面接入电感 L，再接输出电容 C 和负载 R。

第一版参数：

```text
L = 22uH
C = 100uF
Rload = 2.4Ω
```

输出端命名为：

```text
Vout
```

截图位置：

```text
assets/screenshots/02-plecs-output-stage.png
```

### 5. 加入 PWM 信号

开环模型不需要控制器，只需要固定占空比 PWM。

可以使用 Pulse Generator，设置：

```text
Frequency = 200kHz
Duty cycle = 0.5
```

这个 PWM 信号接到高边开关的控制端。

此时模型的逻辑是：

```text
PWM 固定 duty
-> 开关管周期性导通/关断
-> Buck 功率级自然响应
-> 输出电压由 duty 和 Vin 决定
```

截图位置：

```text
assets/screenshots/02-plecs-pwm-duty.png
```

### 6. 加入测量点和 Scope

为了判断模型是否正确，至少需要观察三个波形：

| 波形 | 作用 |
| --- | --- |
| SW 节点电压 | 判断开关管和 PWM 是否工作 |
| 电感电流 IL | 判断电感电流是否连续、纹波是否合理 |
| 输出电压 Vout | 判断输出是否接近理论值 |

建议 Scope 中至少放入：

```text
Vsw
IL
Vout
```

## 仿真设置建议

第一版仿真可以先设置：

| 项目 | 建议值 |
| --- | --- |
| 仿真时间 | 5ms - 10ms |
| 初始 duty | 0.5 |
| 输入电压 | 24V |
| 负载 | 2.4Ω |
| 开关频率 | 200kHz |

仿真时间不要太短，否则输出电压还没稳定就结束；也不要一开始设得太长，避免仿真速度变慢。

如果仿真步长可以设置，建议先使用 PLECS 默认设置。等后续出现波形抖动、开关细节看不清楚、仿真速度过慢等问题时，再调整求解器和步长。

## 预期波形

开环 Buck 正常工作时，应该能看到以下现象：

### 1. SW 节点波形

SW 节点应当是一个高低切换的开关波形。

大致特征：

```text
高电平接近 Vin
低电平接近续流路径电压
频率等于 PWM 开关频率
占空比约为 50%
```

截图位置：

```text
waveforms/02-open-loop-vsw.png
```

### 2. 电感电流 IL

电感电流应该呈现上升和下降的锯齿形纹波。

开关管导通时：

```text
电感电流上升
```

开关管关断时：

```text
电感电流下降
```

如果参数合理，满载下电感电流应当大致围绕 5A 上下波动，而不是直接降到 0A。

截图位置：

```text
waveforms/02-open-loop-il.png
```

### 3. 输出电压 Vout

输出电压应当逐渐上升并稳定到接近 12V。

如果使用理想器件和 duty=0.5，稳态输出应接近：

```text
Vout ≈ 12V
```

如果输出明显偏离 12V，需要先检查：

```text
Vin 是否为 24V
duty 是否为 0.5
负载是否为 2.4Ω
电感和电容是否接对
二极管方向是否正确
测量点是否接在输出端
```

截图位置：

```text

```

| --- | --- | --- |

## 常见问题

### 1. Vout 不接近 12V

优先检查：

```text
duty 是否设置为 0.5
Vin 是否设置为 24V
负载是否设置为 2.4Ω
电感是否串在 SW 和 Vout 之间
输出电容是否接在 Vout 和 GND 之间
二极管方向是否正确
```

不要一上来就怀疑 PLECS 或求解器。大多数初期问题都是连接错误、参数错误或测量点放错。

### 2. 电感电流直接降到 0

这可能说明 Buck 工作在断续电流模式，或者负载太轻、电感值太小。

可以先检查：

```text
负载电阻是否过大
电感值是否过小
开关频率是否设置正确
```

### 3. 仿真报错或无法运行

优先检查：

```text
是否存在电路悬空
是否缺少 Electrical Reference
是否有理想电压源和理想开关形成不合理回路
开关控制信号是否连接正确
```

### 4. 波形看起来很乱

先不要急着调复杂参数。可以先放慢观察节奏：

```text
只看 Vsw
再看 IL
最后看 Vout
```

调试顺序应该从功率级局部波形开始，而不是直接看最终输出。

本文完成了开环 Buck 功率级的搭建思路。

本篇的核心不是让输出电压“看起来对”，而是证明：

```text
PWM 能驱动开关管
SW 节点有正确开关波形
电感电流有合理纹波
输出电压接近 D * Vin
```

只要这几个条件满足，说明功率级模型基本可信。下一篇可以继续讨论 Buck 电感、电容和开关频率的参数估算，并进一步解释为什么电感值、负载和开关频率会影响输出纹波和动态响应。
