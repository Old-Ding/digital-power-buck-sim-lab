# 占空比限幅和抗积分饱和为什么必须做

## 问题

离散 PI 控制器输出的 duty 不能无限大，也不能无限小。真实电源软件必须限制占空比范围，同时处理积分项，否则启动、负载突变或输入电压变化时，积分器可能长时间饱和，导致恢复慢、过冲大，甚至触发保护。

## 本篇目标

```text
给 duty 设置最小值和最大值
观察没有抗积分饱和时的积分累积
加入条件积分或回算抗饱和
对比修改前后的启动和负载突变波形
明确限幅属于控制器职责，不把保护逻辑塞进 PI
```

## 待补

- duty_min / duty_max 初始选择
- PI 输出限幅前后对比
- 积分项 pi_integrator 可观测化
- 抗积分饱和策略选择
- 与软启动和保护状态机的职责边界
