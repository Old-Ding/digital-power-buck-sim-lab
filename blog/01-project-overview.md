# 【数字电源/MATLAB+PLECS】如何进行 Buck 数字电源仿真（一）Buck 电路介绍与项目准备

Buck 电路是一种常见的 DC-DC 降压型开关电源拓扑，广泛应用于嵌入式设备、电机驱动、工业控制、通信电源以及各类低压供电系统中。Buck 电路通过控制开关管的导通和关断时间，将较高的输入直流电压转换为较低的稳定输出电压。

本教程记录使用 MATLAB/Simulink 和 PLECS 进行 Buck 数字电源仿真所需要的一些基础知识，例如 Buck 基本原理、项目规格、仿真工具分工、控制链路、保护功能以及后续章节安排。后续篇章会具体编写 PLECS 功率级模型、MATLAB 控制脚本、离散 PI 控制器、软启动、保护状态机以及 C 风格控制代码。

## Buck 电路介绍

Buck 电路也叫降压斩波电路，主要作用是将输入直流电压降低到目标输出电压。

理想情况下，Buck 电路的输出电压和占空比之间有如下关系：

```text
Vout = D * Vin
```

其中：

| 名称 | 含义 |
| --- | --- |
| Vin | 输入电压 |
| Vout | 输出电压 |
| D | PWM 占空比 |

例如输入电压为 24V，希望输出 12V，则理想占空比约为：

```text
D = Vout / Vin = 12 / 24 = 0.5
```

也就是说，在理想条件下，占空比为 50% 时，24V 输入可以得到 12V 输出。

需要注意的是，实际电路中会存在 MOSFET 导通损耗、二极管压降、电感直流电阻、电容 ESR 等因素，因此实际占空比和理论值会有一定偏差。

## Buck 电路基本组成

典型 Buck 电路主要由以下部分组成：

| 模块 | 功能 |
| --- | --- |
| 输入电源 Vin | 提供直流输入电压 |
| 开关管 MOSFET | 根据 PWM 信号周期性导通和关断 |
| 二极管或同步 MOSFET | 在主开关关断时为电感电流提供续流路径 |
| 电感 L | 储能并平滑电流 |
| 输出电容 C | 滤除输出纹波，稳定输出电压 |
| 负载 R | 消耗输出功率 |
| 采样电路 | 采集输出电压、电流等反馈量 |
| 控制器 | 根据反馈量调整 PWM 占空比 |

在后续 PLECS 模型中，功率级主要会包含输入电压源、MOSFET、续流器件、电感、电容、负载以及测量模块。

![Buck 降压电源基本拓扑](../assets/figures/buck-topology.png)

图中需要重点关注三个位置：

| 位置 | 说明 |
| --- | --- |
| SW 节点 | 开关节点，后续会观察 PWM 开关波形 |
| 电感 L | 观察电感电流是否连续、纹波是否合理 |
| 输出端 Vout | 观察输出电压、输出纹波和负载突变响应 |

## 本系列项目规格

本系列第一阶段先做一个低压 Buck 数字电源仿真项目，不涉及 220V 市电，也不涉及 PFC、LLC、反激等复杂拓扑。

项目规格如下：

| 项目 | 参数 |
| --- | --- |
| 拓扑 | Buck 降压电路 |
| 输入电压范围 | 18V - 30V |
| 标称输入电压 | 24V |
| 输出电压 | 12V |
| 最大输出电流 | 5A |
| 最大输出功率 | 60W |
| 初始开关频率 | 200kHz |
| 控制方式 | 数字 PI 电压环 |
| 仿真工具 | MATLAB/Simulink、PLECS |
| 后续扩展 | C 风格控制代码、STM32G4 或 TI C2000 迁移 |

选择 Buck 作为第一阶段项目，是因为 Buck 电路结构清晰，容易观察占空比、电感电流、输出电压和负载变化之间的关系，同时又能覆盖数字电源控制中的核心内容。

## 为什么先做 Buck

Buck 电路虽然结构简单，但它包含了数字电源软件开发中的很多关键问题。

例如：

```text
如何产生 PWM 信号
如何采集输出电压
如何根据误差调节占空比
如何限制最大占空比
如何实现软启动
如何处理过压、过流、欠压故障
如何观察调试变量
如何将仿真控制器整理成 C 代码
```

这些问题在更复杂的电源拓扑中同样存在。先用 Buck 把这些基础问题解决清楚，再学习 Boost、Buck-Boost、反激、PFC 或 LLC 会更稳。

如果一开始直接学习复杂拓扑，很容易被磁性器件、隔离反馈、谐振参数、驱动保护等内容分散注意力，反而不容易看清数字控制软件的主线。

## 数字 Buck 控制链路

数字 Buck 电源的控制过程可以理解为：

```text
采样输出电压
计算输出误差
PI 控制器计算占空比
更新 PWM
Buck 功率级响应
再次采样输出电压
```

如果输出电压低于目标值，控制器会增大占空比；如果输出电压高于目标值，控制器会减小占空比。

下图是数字 Buck 电源的基本闭环控制链路：

![数字 Buck 电源闭环控制链路](../assets/figures/digital-control-loop.png)

例如目标输出电压为 12V：

| 实际输出电压 | 控制动作 |
| --- | --- |
| Vout < 12V | 增大 duty |
| Vout > 12V | 减小 duty |
| Vout ≈ 12V | 保持或小幅调整 duty |

但真实数字电源软件不能只实现这一个闭环，还需要考虑启动、保护、故障锁存和调试观测。

因此本项目后续会加入如下功能：

```text
软启动
占空比限幅
抗积分饱和
输入欠压保护 UVLO
输出过压保护 OVP
输出过流保护 OCP
过温保护 OTP
故障状态机
关键变量记录
```

## 软件状态机设计

为了让控制逻辑更接近真实工程，本项目不会把所有判断都写在一个控制器函数中，而是采用状态机方式组织。

初步状态如下：

| 状态 | 功能 |
| --- | --- |
| INIT | 初始化控制参数 |
| IDLE | 等待启动命令 |
| SOFT_START | 软启动过程 |
| RUN | 正常闭环运行 |
| FAULT_LATCH | 故障锁存 |
| RECOVERY | 故障恢复 |

状态切换关系如下图所示：

![数字电源软件状态机](../assets/figures/power-state-machine.png)

其中 PI 控制器只负责计算占空比，保护模块负责检测故障，状态机负责决定当前电源应该处于哪种运行状态。

这样设计的好处是职责清晰，后续出现过压、过流或者启动异常时，更容易定位问题。

## 关键观测变量

在仿真和调试过程中，不能只看输出电压，还需要观察多个变量。

本项目重点观察以下变量：

| 变量 | 作用 |
| --- | --- |
| Vin | 输入电压 |
| Vout | 输出电压 |
| Iout | 输出电流 |
| IL | 电感电流 |
| duty | PWM 占空比 |
| Vref | 输出电压参考值 |
| state | 当前电源状态 |
| fault_code | 故障码 |
| pi_integrator | PI 积分项 |

这些变量后续会用于分析启动过冲、负载突变、占空比饱和、过流保护误触发等问题。
