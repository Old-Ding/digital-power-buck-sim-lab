# Buck 电感、电容和开关频率怎么初步选择

## 问题

开环模型跑通后，下一步不能直接跳到 PI。需要先解释电感、电容、开关频率和负载之间的关系，否则后面看到纹波、启动过冲或负载突变，很难判断是功率级参数问题还是控制器问题。

## 本篇目标

```text
根据目标输出电流估算电感纹波
根据输出纹波目标估算输出电容
解释开关频率对纹波、损耗和仿真速度的影响
用 PLECS 扫描 L/C/fsw，观察波形变化
形成一组后续闭环控制可用的基准参数
```

## 待补

- 电感纹波公式推导
- 输出电容纹波估算
- 22uH / 100uF / 200kHz 的选择理由
- 不同电感值下的 IL 纹波对比
- 不同电容值下的 Vout 纹波对比
- 启动过冲和 LC 参数的关系
