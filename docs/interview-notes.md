# 面试讲解稿

## 项目一句话

基于 PLECS 和 MATLAB/Simulink 搭建 24V 输入、12V/5A 输出的数字 Buck 电源仿真项目，并实现离散 PI 电压环、软启动、保护状态机、C 风格控制代码和测试矩阵。

## 必须能讲清楚的问题

- 为什么第一阶段选择 Buck，而不是 PFC、LLC 或反激？
- PI 控制器、保护检测、状态机为什么要分层？
- 软启动解决了什么问题？
- 占空比限幅和抗积分饱和为什么必须做？
- OCP/OVP/UVLO 同时出现时如何确定故障优先级？
- 负载突变时如何判断问题来自 PI 参数、功率级参数还是限幅策略？
- 如何把仿真控制器迁移到 MCU？
